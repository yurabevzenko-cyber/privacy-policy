Privacy Policy

Application name:Ширяївське УЕГГ “для Звіту працівників”

This Privacy Policy explains how the application collects, stores, and processes data using local storage and Firebase services.

1. Data Storage

This application uses a hybrid architecture:

  • Local storage (SQLite / Room database) on the device
  • Cloud services provided by Firebase

2. Local Data

The application stores the following data locally on the device:

  • Customer personal information (name, phone number, address)
  • Gas meter and technical data
  • Service records and inspection data

3. Firebase Services

The application uses Google Firebase for the following purposes:

  • Firebase Authentication (user login and access control)
  • Cloud Firestore / Realtime Database (reporting and statistics)
  • Firebase Cloud Messaging (push notifications)

4. Data Collected by Firebase

Firebase may process the following data:

  • User email and authentication ID (UID)
  • Device notification tokens (for push notifications)
  • Application usage statistics (reports and activity data)

5. Purpose of Data Processing

Data is used only for:

  • User authentication and access management
  • Generating and storing service reports
  • Displaying statistics of performed work
  • Sending system notifications

6. Data Sharing

We do not sell or share personal data with third parties.

Data is accessible only to authorized users of the system.

7. Push Notifications

This application uses Firebase Cloud Messaging to send notifications about updates, reports, and system events.

Device notification tokens are used only for delivering messages.

8. Data Security

Data is protected using Firebase security rules and device-level security mechanisms.

9. Data Deletion

Users may request deletion of their data or remove it by contacting the administrator or clearing local application data.

10. User Access

This application is intended for internal use by authorized personnel only.
